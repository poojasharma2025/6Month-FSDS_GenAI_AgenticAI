Pedestrian Detection with OpenCV

What We Did

  • Built a pedestrian detection system using Haar Cascades and HOG + SVM.
  • Implemented in OpenCV for real-time video and image analysis.
  • Tested on multiple environments with varying conditions.

Why HOG + SVM

  • HOG captures shape and edge features of pedestrians.
  • SVM provides robust classification based on extracted features.
  • Balanced trade-off between accuracy and speed for detection.

Demo Output

  • Detected pedestrians in still images and live video streams.
  • Worked across different camera angles and lighting conditions.
  • Highlighted bounding boxes around detected people.

Limitations

  • Performance drops in crowded or occluded scenes.
  • Struggles under poor lighting or extreme viewing angles.
  • Not as fast as modern deep learning approaches.

Future Work

  • Upgrade system with YOLO for real-time deep learning detection.
  • Leverage GPU acceleration for higher speed and efficiency.
  • Expand to autonomous driving and smart surveillance use cases.
